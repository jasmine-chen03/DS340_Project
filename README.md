# DS340 Project
### ***Research Question:***
How does the time spent on unpaid childcare differ between single parents (households with only one adult and at least one child) and parents with other members in their household in 2024, and how does this relationship vary by gender, education, religious activity, and region?

### ***Data Source***
**American Time Use Survey (2024): https://www.bls.gov/tus/**
atusact_2024.dat → Activity file (childcare and work activities)
atusresp_2024.dat → Respondent-level demographics
atusrost_2024.dat → Household roster (to identify children, spouses/partners)
atuswho_2024.dat, atuscps_2024.dat → additional demographics (region, education, religion)
