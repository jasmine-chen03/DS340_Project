# DS340 Project
### ***Research Question:***
How does the time spent on unpaid childcare differ between single parents (households with only one adult and at least one child) and parents with other members in their household in 2024, and how does this relationship vary by employment, gender, education, and region? How has the trend changed over time since 2003? <br>

### ***Data Source***
**American Time Use Survey (2024): https://www.bls.gov/tus/** <br>
atusact_2024.dat, atusact_0324.dat → Activity file (childcare and work activities) <br>
atusresp_2024.dat, atusresp_0324.dat → Respondent-level demographics <br>
atusrost_2024.dat, atusrost_0324.dat → Household roster (to identify children, spouses/partners) <br>
atuswho_2024.dat, atuscps_2024.dat, atuswho_0324.dat, atuscps_0324.dat → additional demographics (region, education, religion)
